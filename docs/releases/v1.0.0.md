# v1.0.0 — Initial Stable Release

Playwright website scraping toolkit with Express backend and desktop GUI options.

## Highlights

- Multi-page scrape, asset download, screenshot capture
- Compliance and responsible-use section
- Windows-friendly quick start

## Install

```powershell
git clone https://github.com/mafzalkalwardev/playwright-website-scraper-pro.git
cd playwright-website-scraper-pro
npm install
npm start
```

Use authorized demo URLs only.
